# Network-Packet-Analyzer
A simple packet analyzer built using Python and Scapy.

Features
Captures network packets
Displays Source IP
Displays Destination IP
Detects Protocol (TCP, UDP, ICMP)
Displays Packet Size
Requirements
Python 3.x
Scapy
Npcap (Windows)
Installation
pip install -r requirements.txt
Run
python packet_analyzer.py
Sample Output
Source IP: 192.168.1.10
Destination IP: 8.8.8.8
Protocol: UDP
Packet Size: 60 Bytes
